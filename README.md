NETLIFY: https://weather-app-react-11.netlify.app/

A modern, responsive weather dashboard built with **React**.

## Features
- **Real-time Data:** Fetches current weather using the Open-Meteo API.
- **Detailed Forecasts:** Includes hourly breakdown and a 7-day daily forecast.
- **Unit Conversion:** Toggle between Celsius/Fahrenheit, km/h / mph, and mm/in.
- **City Search:** Instant search for any location worldwide.
- **Responsive Design:** Optimized layout using CSS Grid and Flexbox.

## Tech Stack
- React.js
- CSS3 (Variables, Grid, Flexbox)
- Open-Meteo API
- Vite
